欢迎来到国琼大药房后台管理系统！

       国琼大药房致力于社区半小时精细化服务的提升，结合人工智能技术，实现在线就医，半小时内药品送货上门服务等精细化服务，并且对每人建立实时在线的医疗档案，解决往期病史、往期用药以及往期禁用药的不清晰问题，做到了手机购药时的实时提醒。通过大数据分析和算法，实现本药店药品价格为全国最低。通过自动化检测管理系统和专人专项检测的方式，实现了近效期药品数量为零，不合格药品为零。通过自动化采购系统和自动化舆情检测系统，做到了药店不缺货不缺品中，做到了药品不缺质量。因此国琼大药房历经20年，两代人的努力，终于为大家打造出了一个用药省心、用药方向、用药安心的手机在线购物平台。

       半小时精细化服务内容,如下:

  1. 人工智能辅助病情检测+在线药师：通过线上人脸识别以及舌部的颜色、形状，人工智能可以初判病情。再结合在线执业医师，实现足不出户，在家就医！
  2. 半小时送达：从手机支付后开始计算时间，半小时内将药品送达，超出时间免费
  3. 双码确认：配送人员取药时需要扫码，送到时客户需要扫码，将送错要的几率降为零
  4. 服药时间提醒：自动设置用药时间提醒，提醒的方式为电话语音
